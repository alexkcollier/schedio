# schedio-design

## Requirements

- Node 8.11.4+
- @vue/cli 3.0.1

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
# Vue app localhost:8080
npm run serve

# Storybook localhost:6006
npm run serve:storybook
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
